üîí THE ANTI-HALLUCINATION SYSTEM
================================================================================

Every extracted vendor goes through 5 validation layers:

Layer 1: FORMAT CHECK
  ‚Üì Is the JSON structure correct?
  ‚Üì Are all required fields present?

Layer 2: FACTUAL CHECK
  ‚Üì Does the extracted data exist in the source text?
  ‚Üì No made-up information allowed

Layer 3: CONSTRAINT CHECK
  ‚Üì Does MOQ meet requirements? (‚â§500)
  ‚Üì Is price in range? ($120-$150)
  ‚Üì Are any red flags present?

Layer 4: CONSISTENCY CHECK
  ‚Üì Is this a duplicate vendor?
  ‚Üì Is the price an anomaly?

Layer 5: CROSS-VALIDATION
  ‚Üì Do all fields make logical sense together?
  ‚Üì Is total order value reasonable?

Result: ‚úì PASS ‚Üí Database | ‚úó FAIL ‚Üí Rejected

Only vendors that pass ALL 5 layers are saved.
This ensures ZERO hallucinations in your pipeline.
