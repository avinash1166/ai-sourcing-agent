## Learning Context for AI

### Price Learning
```
REFERENCE PRODUCT:
- Vendor: HYY Technology
- Base price: $95-160
- Target: $70-90
- Strategy: Find competitors with similar specs at lower price OR negotiate volume discount

PRICE NEGOTIATION RULES:

2. Mention "evaluating multiple suppliers" to create competition
3. Be transparent about target price range ($70-90)
4. Willing to pay for sample at higher price if bulk is good
5. Factor in shipping (FOB China vs DDP)
```

### Specification Learning
```
MUST HAVE:
- 15.6" touchscreen (capacitive, not resistive)
- Android OS (version 8+)
- Front camera
- WiFi + 4G/LTE capability
- Removable/customizable casing

NICE TO HAVE:
- eSIM support
- Higher resolution (2K/4K)
- Better CPU (RK3588)
- More RAM (4GB)
- NFC support

DEALBREAKERS:
- Battery required (we want plug-in only)
- Windows-only OS
- No customization allowed
- MOQ > 500
- Price > $100 even at 500 units
```
